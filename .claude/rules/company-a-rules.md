# 회사 A 운영 규칙

## 언어

모든 내부 문서와 결과물은 한국어로 작성한다.
코드, Schema, API 필드명은 영어를 사용한다.

## 기밀 처리

- 회사 A의 고객명, 계약 내용, 재무 수치를 외부 API에 직접 전달하지 않는다.
- 외부 전달이 필요한 경우 반드시 사람 승인을 받는다.

## 결과물 기준

- 모든 결과물에는 사용한 Source의 참조(Drive 파일 ID 또는 Episode ID)를 포함한다.
- 출처가 없는 주장은 "[출처 불명]", "[가정]", "[추정]" 으로 명시한다.

## GBrain 사용

- 프로젝트 시작 전 반드시 관련 Episode를 조회한다.
- 프로젝트 종료 후 반드시 Episode를 생성하고 저장한다.

## Google Drive 사용

- 현재 Read-only. Drive 파일 수정은 사람이 직접 한다.
- Drive 파일 ID와 URL을 Source Manifest에 기록한다.
